# Parquímetro - JavaScript POO

Aplicação simples que simula o funcionamento de um **parquímetro**, desenvolvida com **HTML, CSS e JavaScript**, utilizando **Programação Orientada a Objetos (POO)**.

O usuário informa um valor em reais e a aplicação retorna o **tempo de permanência permitido** e o **troco**, quando houver.

---

## Funcionalidades

- Receber valor informado pelo usuário
- Calcular o tempo de estacionamento conforme o valor:
  - R$ 1,00 → 30 minutos
  - R$ 1,75 → 60 minutos
  - R$ 3,00 → 120 minutos (tempo máximo)
- Calcular e exibir o troco
- Exibir mensagem de **valor insuficiente** quando o valor for menor que R$1,00
